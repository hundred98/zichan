# zichan
This is a B / s management system based on LabVIEW. The database is mysql, the front end uses layui, and the back end uses LabVIEW.  At present, most functions have been completed, and basic business management functions can be realized.

这是一个基于LabVIEW开发的B/S管理系统，使用的数据库是MySQL，前端使用的是LayUI，后端使用的是LabVIEW。
目前已经完成了大部分功能，可以实现基本的业务管理功能。

LabVIEW is a voice based on graphics programming. Using this language can greatly improve the development efficiency, thus shorten the development time, which can be reduced to 20% of the development time of ordinary voice skillfully.

At the same time, this development language can really realize business oriented visual programming, which can pay more attention to business process implementation.

Compared with relying on more frameworks, it can reduce the time of learning frameworks and the difficulty of getting started.

What you see is what you get. You can see the data flow and business process of the whole business at a glance through the program.

LabVIEW是基于图形编程的语音，使用该语言可以极大地提高开发效率，从而缩短开发时间，熟练地情况下可以将开发时间缩短到普通语音开发的20%。
同时这种开发语言可以真正实现面向业务的可视编程，从而可以更关注业务流程实现。
相比依赖更多框架，可以减少更多学习框架的时间，降低了入门难度。
所见即所得，可以通过程序一目了然的看清楚整个业务的数据流和业务流程。
